# SuwolView 0.2.1

Patch release after 0.2.0.

## Highlights

- Linux AppImage update-ready release path.
- Linux tar.gz portable archive.
- `latest-linux.yml` update metadata.
- `checksums.txt` and detached GPG signature.
- Public release key included.
- Update settings and manual update checks.
- Safe Mode disables update checks.
- Stability, logging, settings recovery, and cache maintenance improvements.
- New PNG-based app icon source and regenerated Windows icons.

## Downloads

- Windows installer: `SuwolView-0.2.1-setup.exe`
- Windows portable: `SuwolView-0.2.1-win-x64.zip`
- Linux AppImage: `SuwolView-0.2.1-linux-x64.AppImage`
- Linux portable: `SuwolView-0.2.1-linux-x64.tar.gz`

## Notes

- v0.2.0 remains the previous public release.
- v0.2.1 is the current update-ready release.
- Windows installer is recommended for normal Windows users.
- Linux AppImage is the intended in-app update path.
- Linux tar.gz is manual update only.
- Automatic update checks are off by default.
- Safe Mode disables update checks.
- macOS builds are not included yet.
